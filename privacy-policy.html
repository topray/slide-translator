개인정보 처리방침

슬라이드 번역기 (Slide Translator) Chrome 확장 프로그램

요약: 본 확장 프로그램은 사용자의 개인정보를 수집·저장·전송하지 않습니다. API 키는 사용자의 기기에만 저장되며, 슬라이드 텍스트는 번역을 위해 Anthropic Claude API로만 전송됩니다.

1. 수집하는 정보

본 확장 프로그램은 사용자의 개인정보를 별도로 수집하지 않습니다. 확장 프로그램이 처리하는 데이터는 다음과 같습니다:

  • Anthropic API 키: 사용자가 직접 입력하며, Chrome의 로컬 저장소(chrome.storage.local)에만 저장됩니다. 외부 서버로 전송되지 않습니다.
  • Google Slides 텍스트: 현재 보고 있는 슬라이드의 텍스트만 추출하며, 번역 요청 시 Anthropic Claude API로 전송됩니다.
  • 번역 결과: 번역된 텍스트는 사용자의 기기 내 로컬 저장소에 캐시됩니다.

2. 데이터 저장

모든 데이터는 사용자의 기기에만 저장됩니다:

  • API 키: chrome.storage.local (기기 내 로컬 저장, 동기화 없음)
  • 번역 캐시: 메모리 내 임시 저장 (브라우저 재시작 시 초기화)
  • 슬라이드별 번역 상태: chrome.storage.local

본 확장 프로그램은 별도의 외부 서버를 운영하지 않으며, 사용자 데이터를 자체 서버에 저장하지 않습니다.

3. 데이터 전송

확장 프로그램이 외부로 전송하는 데이터는 다음 한 가지뿐입니다:

  • 슬라이드 텍스트 → Anthropic Claude API: 번역을 위해 현재 슬라이드의 텍스트가 Anthropic의 API(https://api.anthropic.com)로 전송됩니다. 이 데이터는 Anthropic의 개인정보 처리방침에 따라 처리됩니다.

그 외 어떠한 제3자 서비스에도 데이터를 전송하지 않습니다.

4. 제3자 서비스

본 확장 프로그램은 Anthropic Claude API만을 사용합니다. Anthropic의 데이터 처리에 관한 자세한 내용은 Anthropic 개인정보 처리방침을 참조하시기 바랍니다.

5. 사용자 권리

사용자는 언제든지 다음 작업을 수행할 수 있습니다:

  • 설정 페이지에서 API 키를 삭제할 수 있습니다.
  • 설정 페이지에서 모든 번역 캐시 및 저장 데이터를 삭제할 수 있습니다.
  • 확장 프로그램을 제거하면 모든 로컬 데이터가 자동으로 삭제됩니다.

6. 변경 사항

본 개인정보 처리방침이 변경될 경우, 이 페이지를 통해 공지합니다.
